# JDBC Test Project
This is a test project using JDBC and lombok (as tool).

## How to build
To build execute command:
````
mvn package
````
## How to run
To run execute command:
````
# enter directory with build jar
cd target 
java -jar jdbc_students-1.0-SNAPSHOT.jar
````
